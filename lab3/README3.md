## 1. Мета роботи
Практичне використання команд DML (SELECT, INSERT, UPDATE, DELETE) для імітації реальних операцій ігрового маркетплейсу.

## 2. Опис виконаних запитів

| Тип | Мета запиту | Результат |
| :--- | :--- | :--- |
| **SELECT** | Отримання списку товарів дорожче 50 грн | Виведено перелік преміум-скінів та інструментів |
| **INSERT** | Реєстрація нового гравця та додавання товару | В базу успішно додано користувача `New_Pro_Player` |
| **UPDATE** | Поповнення балансу користувача `Mikhail_Dev` | Поле `wallet_balance` успішно збільшено на 100 одиниць |
| **DELETE** | Скасування замовлення №1 та видалення підозрілої сесії | Рядок з `order_id = 1` успішно видалено з таблиці Orders |
